Miscellaneous updates

  • The Futuramerlin Web site should eventually become a Web site for Futuramerlin again, and Ember moved to a new home.
  • In the unlikely event that anyone is using Wreathe GNU/Linux:
    • Wreathe 7.3 will probably arrive eventually, but it's hard to say when.
    • If anyone wants a relatively normal GNU distribution, I suggest switching away from Wreathe to GNU Guix (or one of these other free distributions if you prefer).
    • Wreathe 8 and later will probably not be a normal GNU distribution, and instead be a DDC environment over a minimal base system. This would presumably not have the same feature set as past Wreathe versions, hence the recommendation to switch to Guix or similar for something similar to these past Wreathe versions.
    • Wreathe 7.3 should include a preliminary version of the DDC environment for testing, in some form or another.
