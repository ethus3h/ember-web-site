Side projects

Page location: Home

These are secondary projects developed in the course of working on Ember that can be used separately.

  • Crystallize, a package of command-line tools for working with files and filesystems: mostly complete
  • ember-shared, a package of command line tools code used by other Ember software: needs polishing
  • Futuramerlin Web Toolkit, a static Web site builder: needs polishing
  • Wreathe, a computing environment: partly complete

Development documentation for Ember in general is available.

See also the list of discontinued projects.
