Guidance-centric user interface for IT systems: a proposal

This document proposes an interface for information technology systems to provide better accessibility for people with dementia and other mental challenges that affect the ability to hold state information, as well as suggesting a unified document editing platform that handles a diverse array of information so as to avoid unnecessary restrictions on combining diverse types of information within a single context.
