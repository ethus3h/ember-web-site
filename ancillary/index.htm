Side projects

Page location: Home

These are secondary projects that are related to the development of the main project. The main project is not yet functioning in a manner such that it can be used, but these ancillary projects are generally usable. (Wreathe was formerly considered ready, but has temporarily been marked unready due to changes in the surrounding information technology environment.)

  • Crystallize, a file archival app: needs polishing
  • ember-shared, code shared between various Ember software: needs polishing
  • Futuramerlin Web Toolkit: needs polishing
  • Wreathe, a computing environment: partly complete
  • Ember Retired Repositories (a project to preserve the history of discontinued Ember projects and no-longer-needed work): partly complete

There are also some projects in development, and development documentation for Ember in general is available.

See also the list of discontinued projects.
