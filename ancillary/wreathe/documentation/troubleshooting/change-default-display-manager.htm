How to change the default display manager? (Aptitude doesn't ask which default display manager to use during configuration?) — Wreathe Documentation

Page location: Software Wreathe Documentation Troubleshooting

This article applies to: Wreathe 7.2

Problem

Don't know how to change the default display manager. How to change the default display manager? (Aptitude doesn't ask which default display manager to use during configuration?)

Solution

Edit /etc/X11/default-display-manager

(useful paths:

  • gdm = /usr/sbin/gdm
  • kdm = /usr/bin/kdm
)
