How to make a video from a still picture and a sound recording — Wreathe Documentation

Page location: Software Wreathe Documentation Documentation for experimental features

This article applies to: Wreathe 7.3

Using VLC's "Convert / Save" wizard, use the "Play another media synchronously" option (under the "Show more options" checkbox) to play the image file. Starting with an existing MP3 audio file (with the "Keep original audio track" option enabled) seems to work best for creating an MP4 video with MP3 audio, rather than trying to transcode the audio at the same time as creating the MP4.
