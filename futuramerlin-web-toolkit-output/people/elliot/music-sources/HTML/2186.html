Calculator.

Calculator is a simple calculator for Microsoft Windows. It is compiled using Microsoft .NET, so it probably will not run using WINE/Darwine. Please feel free to e-mail me with any questions you wish to ask. A known bug is that it gives an incorrect answer when calculating the square root of a negative number. This will be fixed in the next release.

Functions (of the latest version):

Addition
Subtraction
Divsion
Multiplication
Exponents
Integer division
Modulus division (presents answer as a remainder)
Area calculations
Perimetre calculations
Temperature conversion
Prime verification
Square Root
Scratch pad
Lockable sessions
Dual-method input
Random number generation
