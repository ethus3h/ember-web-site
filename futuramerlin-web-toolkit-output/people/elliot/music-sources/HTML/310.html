Jungle Theme for Preview

  Jungle Theme for Preview is a package that redraws Preview's icon set.
  To install it, copy the files in the PKZip file to /Applications/
  Preview.app/Contents/Resources/. This can be achieved by navigating
  to /Applications/ in Finder, then, while holding the control key, clicking
  Preview. Please feel free to e-mail me with any questions you wish to ask.
  ask. The colours are from GIMP's Tropical Colors gradient.
