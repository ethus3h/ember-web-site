Blog Post: Bug in Preview — 4 December, 2009

Preview seems reluctant to let me export AppleMenuIcon.icns (/System/Library/CoreServices/CoreTypes.bundle/AppleMenuIcon.icns) as a PNG. My steps for attempting to do so were as follows.

  1. Open the image in Preview.
  2. Press ⇧-⌘-S to get the Save As... option.
  3. Select "PNG" from the option box.
  4. Click "Save". This doesn't work!
  5. Click "OK". Back to where I started.
  6. Press ⇧-⌘-S to get the Save As... option.
  7. Select PNG from the option box.
  8. Select "8 Bits/Channel" from the option box. Maybe this will work better.
  9. Click "Save". This still doesn't work.

To convert this, I ended up using GraphicConverter, but unfortunately, it lost the alpha channel. I don't know why Preview couldn't convert it. I would like to know why Preview couldn't convert it. If you know, please e-mail me.
