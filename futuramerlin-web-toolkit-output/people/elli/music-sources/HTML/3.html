Calculator.

  Calculator is a versatile Calculator application. It is compiled via
  Microsoft .NET, so it probably will not run using WINE. It has many
  standard calculator features. It has addition, subtraction, multiplication,
  division, mod division, area, perimeter, Celsius to Fahrenheit and vice
  versa, documentation, and prime verification. Please feel free to e-mail
  me with any questions you wish to ask.
