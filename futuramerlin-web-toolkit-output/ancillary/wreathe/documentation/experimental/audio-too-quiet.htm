Sound is quieter than expected — Wreathe Documentation

Page location: Software Wreathe Documentation Documentation for experimental features

This article applies to: Wreathe 7.3

If sound is quieter than you want, even at the maximum volume, follow the following steps:

  1. Stop PulseAudio by running pulseaudio --kill.
  2. Open Alsamixer, e.g. by running alsamixer.
  3. In Alsamixer, set the Master and output (e.g. "Headphone") channels to 100%.
  4. Close Alsamixer (when using the terminal version, press escape to close it).
  5. Restart PulseAudio by running pulseaudio --start.
