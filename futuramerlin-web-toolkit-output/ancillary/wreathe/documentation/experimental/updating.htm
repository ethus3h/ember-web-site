How to update Wreathe — Wreathe Documentation

Page location: Software Wreathe Documentation Documentation for experimental features

This is an article for an experimental, unfininshed version of Wreathe.

This article applies to: Wreathe 7.3

To update an installation of the development version:

  • Update wreathe-base and the package databases: smart-live-rebuild -f app-misc/wreathe-base -f app-misc/wreathe-overlays-redist; layman -S; eix-update
  • Update upstream's ebuilds: emerge --update app-misc/gentoo-portage-tree; eix-update
  • Update all other live ebuilds: smart-live-rebuild
  • Update all other packages: emerge --backtrack=0 --deep --complete-graph=y --with-bdeps=y --changed-deps=y --newuse --update --verbose --verbose-conflicts @world
  • Rebuild packages for any newly built libraries: emerge @preserved-rebuild
  • Rebuild Xorg and kernel modules: emerge @x11-module-rebuild @module-rebuild
  • Update Haskell packages: haskell-updater
