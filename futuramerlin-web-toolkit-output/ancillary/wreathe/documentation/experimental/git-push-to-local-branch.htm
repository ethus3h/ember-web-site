Git: How to (force) push from one local branch to another local branch? ("Push" is the wrong word.) — Wreathe Documentation

Page location: Software Wreathe Documentation Documentation for experimental features

This article applies to: Wreathe 7.3

Before doing this, make sure you have all changes saved somewhere safe (uncommitted changes will be overwritten), and make a new branch from the destination branch (git checkout destinationBranchName; git branch newBranchName) (commits not in the source branch will be overwritten): git checkout destinationBranchName; git reset --hard sourceBranchName, replacing the branch names. This will make destinationBranchName point to the same commit as sourceBranchName.
