Permission denied when emerging, despite being root — Wreathe Documentation

Page location: Software Wreathe Documentation Troubleshooting

Table of Contents

  1. Problem
  2. Possible solution

This article applies to: Wreathe 7.3

Problem

Despite being root, attempting to use emerge gives "Permission denied" errors.

Possible solution

Ensure that the permissions are set correctly on your Portage tree and overlays: they should be owned by the user portage and the group portage (e.g. chown -R portage:portage /example).
