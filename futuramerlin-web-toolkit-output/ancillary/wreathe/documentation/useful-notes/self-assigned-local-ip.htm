Request a self-assigned local IP address — Wreathe Documentation

Page location: Software Wreathe Documentation Useful notes

This article applies to: Wreathe 7.2

To request a self-assigned local IP address, run the command

sudo avahi-autoipd --force-bind eth0


, substituting what fits your use case for "eth0".
