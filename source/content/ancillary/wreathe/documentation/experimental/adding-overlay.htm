How to add an overlay — Wreathe Documentation

Page location: Software Wreathe Documentation Documentation for experimental features

Using unsupported overlays could break your system.

This article applies to: Wreathe 7.3

Some overlays are included with the system. Run layman -L to get a list of these available overlays. They can be added by running layman -a overlay-name, substituting the desired overlay name.

To add other overlays run layman -o https://example.org/foo.xml -f -a overlay-name, substituting the desired overlay's name and definition file URL.
