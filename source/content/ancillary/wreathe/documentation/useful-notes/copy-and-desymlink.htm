Copy all symlinks in symdir to symdir_output, but change them into the target files — Wreathe Documentation

Page location: Software Wreathe Documentation Useful notes

This article applies to: Wreathe 7.2

Copy all symlinks in symdir to symdir_output, but change them into the target files (replace them with the real linked files):

rsync symdir/ symdir_output/ -a --copy-links -v
