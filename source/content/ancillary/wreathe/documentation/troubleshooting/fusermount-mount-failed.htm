fusermount: mount failed: Operation not permitted — Wreathe Documentation

Page location: Software Wreathe Documentation Troubleshooting

This article applies to: Wreathe 7.2

Problem

When trying to mount using sshfs, an error is given:

fusermount: mount failed: Operation not permitted

Solution

Run:

# chmod 4755 /usr/bin/fusermount
