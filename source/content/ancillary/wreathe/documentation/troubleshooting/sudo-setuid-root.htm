Trying to use sudo says it must be setuid root — Wreathe Documentation

Page location: Software Wreathe Documentation Troubleshooting

This article applies to: Wreathe 7.2

Problem

Trying to use sudo says it must be setuid root

Error is like this:

sudo: must be setuid root

Solution

To fix, run:

# chown root:root /usr/bin/sudo
    # chmod 4755 /usr/bin/sudo
