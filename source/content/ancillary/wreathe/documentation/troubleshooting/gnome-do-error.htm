Gnome-Do error message: "Wnck-CRITICAL **: wnck_set_client_type got called multiple times." — Wreathe Documentation

Page location: Software Wreathe Documentation Troubleshooting

This article applies to: Wreathe 7.2

Problem

Gnome-Do gives an error message output similar to this to the terminal when run:

(Do:11035): Wnck-CRITICAL **: wnck_set_client_type got called multiple times.

Solution

This message is apparently normal (see the bug report) and can "safely be ignored" (source).

If you came here because your Gnome-Do keyboard shortcut runs Gnome-Do twice, see the troubleshooting solution for that issue instead.
