 Encoding:

This encoding is hierarchal in that it does not have a fixed set of codepoints, but is infinitely extensible, as desired. Each character requires 8 bits. A character can either be a character in the traditional sense, or a character that redefines the encoding to another preset encoding. This technique makes for significantly smaller files, while increasing the diversity of possible encodings.
